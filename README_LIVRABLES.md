## Export PDF

Les PDF peuvent etre regeneres avec :

```powershell
python .\tools\export_markdown_to_pdf.py
```

Le script principal utilise `markdown` et Chrome/Edge headless. Si le headless echoue, `tools/export_markdown_to_pdf_pillow.py` regenere les PDF avec PIL sans dependance externe.
